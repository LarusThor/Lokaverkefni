# Lokaverkefni

Hér skráum við öll helstu tól sem við þurfum að nota við gerð verkefnisins.

Version control: GitHub
  Vantar
  - .gitignore (fyrir það umhverfi sem við ætlum okkur að nota)
  - Branch protection reglur (koma í veg fyrir conflicts)
  - Koma saman um form fyrir commit skilaboð

Aðferðafræði: Agile?
  Vantar
  - Negla niður aðferðafræði sem hentar verkefninu

Task management: Trello?
  Vantar
  - Uppsetning
  - Skrá niður kröfur

Documentation: Overleaf?
  Vantar
  - Ákveða hvað við ætlum að nota fyrir skýrsluna
  - Læra á það tól

Fyrstu skref: Plana
  Vantar
  - Leggja fram plan fyrir önnina (hvað er unnið hvenær)
  - Hvenær er unnið í verkefninu og hvar?
  - Hvenær viljum við hitta Gumma?
  - Rannsóknarvinna (hvaða þróunnarumhverfi eigum við að nota)
  - Þegar við höfum lagt fram plan í grófum drætti: greining og hönnun
